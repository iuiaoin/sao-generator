# sao-generator

> Generators of sao scaffold.

## Usage

Install [SAO](https://github.com/saojs/sao) first.


### From npm

```bash
// 'preplan' is a branch name without 'sao-' which means a type of generator. It is preplan here.
sao preplan my-project    
```
